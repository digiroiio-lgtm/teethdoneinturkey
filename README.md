# teethdoneinturkey

A Next.js website for Teeth Done in Turkey — affordable dental treatments including veneers, implants, crowns and more.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Contact Form

The `/contact` page uses [Formspree](https://formspree.io) (form ID `maqawywr`) via the `@formspree/react` package for handling enquiries.

## Build

```bash
npm install --prefer-offline && npx next build
```
